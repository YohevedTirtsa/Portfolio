Power 4

  • Realization of a Power 4 game in console application mode.
  • When the program starts, a menu should appear offering the following options: Play, Test mode, Exit.
  • 1. Play: This is what starts the game with two human players.
  • 2. Test mode; This is a mode where all the unit tests for the application are run.
  • 3. Exit: Application end
  • Date : 2021
  • Skills : C, C++
  • Softwares : CodeBlocks, GTest
